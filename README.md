# React + TypeScript + Vite

## Version

- Node 20.12.2
- React 18.3.1
- Vite 6.0.5
- yarn 4.5.3

## 배포 주소

[배포](https://yune-inc.vercel.app)

## 실행

```bash
yarn dev
```

## 사용한 라이브러리

- Recoil
- TailWindCSS
- Drag & Drop: @hello-pangea/dnd
- file-saver
- html2canvas
- uuid

## 주요 기능

- 재귀 컴포넌트로 무한확장 가능한 그룹기능
- 그룹내 dnd 정렬 기능
- 그룹 내 요소 추가 기능
- 그룹 png 다운로드 기능
